三角関数​

皆さん、子供の頃に三角関数を学んだことがあると思いますが、ほとんどの方が三角関数の達人ではないでしょうか。

三角関数は物理学や工学の分野で広く使用されており、特に波動や振動の記述に役立ちます。

波とは本質的にエネルギーの伝達方法であり、この伝達は通常「周期的な振動」という形で現れます。

周期的？

よく知られている例として振り子があります。振り子が左右に動く過程は、周期的な運動の一例です。その運動軌跡を観察すると、位置が波のように変化し、正の値からゼロ、次に負の値に変わり、再び繰り返されるのがわかります。

そのため、波をより正確に記述するために三角関数を使用し、数式で表すと次のようになります：

y(t)=Asin⁡(2πft)y(t) = A \sin(2\pi f t)y(t)=Asin(2πft)

ここで、各変数は振り子の運動に関係しています：

  • y(t)y(t)y(t)：時刻 ttt における振り子の位置。
  • AAA：振幅、振り子が振れる最大の距離。
  • fff：周波数、振り子が 1 秒間に何回振れるか。

正弦関数を使用することで、波の特性をより正確に記述できます。

もしよく観察すると、いくつかの波はゼロから始まるのではなく、少し遅れて振動を開始することがあります。この遅れは「位相 ϕ\phiϕ」で表されます。

波形の完全な式は次のように書けます：

y(t)=Asin⁡(2πft+ϕ)y(t) = A \sin(2\pi f t + \phi)y(t)=Asin(2πft+ϕ)

または、余弦関数を使って表現することもできます：

y(t)=Acos⁡(2πft+ϕ)y(t) = A \cos(2\pi f t + \phi)y(t)=Acos(2πft+ϕ)

正弦波と余弦波の違いは位相にあり、どちらも波の変化を記述するために使用できます。

さらに、波の変化の速度をより詳細に記述するために、「角速度 ω\omegaω」を使用できます。これは、波が 1 秒間にどれだけ位相を変化させるかを示し、次のように表されます：

ω=2πf\omega = 2\pi fω=2πf

複素数の形式​

正弦波と余弦波の三角関数表示は非常に直感的ですが、数学や工学の応用では、波形を複素数で表現する方が簡潔で強力です。

なぜでしょうか？

それは、複素数を使うことで正弦波と余弦波の組み合わせを 1 つの式に統一でき、また重ね合わせや微分、積分などの計算が便利に行えるからです。

複素数の形式を理解する前に、まず複素数の基本的な概念を簡単に復習しましょう。複素数は実部と虚部で構成され、次のように表されます：

z=a+biz = a + biz=a+bi

ここで：

  • aaa は実部。
  • bbb は虚部。
  • iii は虚数単位で、i2=−1i^2 = -1i2=−1を満たします。

複素数はまた、「極座標形式」で表すこともできます：

z=r(cos⁡θ+isin⁡θ)z = r(\cos\theta + i\sin\theta)z=r(cosθ+isinθ)

ここで：

  • r=a2+b2r = \sqrt{a^2 + b^2}r=a2+b2​ は複素数の絶対値で、複素数が原点から点 (a,b)(a, b)(a,b) までの距離を示します。
  • θ=tan⁡−1(b/a)\theta = \tan^{-1}(b/a)θ=tan−1(b/a) は複素数の偏角で、実軸の正方向から反時計回りに回転した角度です。
  • cos⁡θ\cos\thetacosθ と sin⁡θ\sin\thetasinθ は角度 θ\thetaθ に対応する方向を定義します。

直交座標系で点 (a,b)(a, b)(a,b) は、極座標系で (r,θ)(r, \theta)(r,θ) として表現できます。

rrr と θ\thetaθ を三角関数に代入すると、z=r(cos⁡θ+isin⁡θ)z = r(\cos\theta + i\sin\theta)z=r(cosθ+isinθ) となります。

オイラーの公式​

数学的定義から見ると、指数関数 exe^xex は無限級数として展開できます：

ex=1+x1!+x22!+x33!+⋯e^x = 1 + \frac{x}{1!} + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdotsex=1+1!x​+2!x2​+3!x3​+⋯

この級数は xxx が実数のときには直感的ですが、xxx が複素数の場合にも適用できます。

例えば、x=iθx = i\thetax=iθ のとき、式に代入すると次のようになります：

eiθ=1+iθ1!+(iθ)22!+(iθ)33!+⋯e^{i\theta} = 1 + \frac{i\theta}{1!} + \frac{(i\theta)^2}{2!} + \frac{(i\theta)^3}{3!} + \cdotseiθ=1+1!iθ​+2!(iθ)2​+3!(iθ)3​+⋯

(iθ)n(i\theta)^n(iθ)n を展開すると、i2=−1,i3=−i,i4=1i^2 = -1, i^3 = -i, i^4 = 1i2=−1,i3=−i,i4=1 などの周期性が現れます。よって、eiθe^{i\theta}eiθ の展開は実部と虚部に分けることができます。

実部の項は次のようになります：

1+(iθ)22!+(iθ)44!+⋯=1−θ22!+θ44!−⋯1 + \frac{(i\theta)^2}{2!} + \frac{(i\theta)^4}{4!} + \cdots = 1 - \frac{\theta^2}{2!} + \frac{\theta^4}{4!} - \cdots1+2!(iθ)2​+4!(iθ)4​+⋯=1−2!θ2​+4!θ4​−⋯

これはちょうど cos⁡θ\cos\thetacosθ の展開式に一致します。同様に、虚部の項は次のようになります：

(iθ)1!+(iθ)33!+(iθ)55!+⋯=i(θ−θ33!+θ55!−⋯ )\frac{(i\theta)}{1!} + \frac{(i\theta)^3}{3!} + \frac{(i\theta)^5}{5!} + \cdots = i\left(\theta - \frac{\theta^3}{3!} + \frac{\theta^5}{5!} - \cdots\right)1!(iθ)​+3!(iθ)3​+5!(iθ)5​+⋯=i(θ−3!θ3​+5!θ5​−⋯)

これもまさに sin⁡θ\sin\thetasinθ の展開式に一致します。

したがって、実部と虚部を総合すると、あの有名なオイラーの公式が得られます：

eiθ=cos⁡θ+isin⁡θe^{i\theta} = \cos\theta + i\sin\thetaeiθ=cosθ+isinθ

ここで：

  • eiθe^{i\theta}eiθ は複素数の指数表示で、実部 cos⁡θ\cos\thetacosθ と虚部 isin⁡θi\sin\thetaisinθ を含んでいます。
  • 複素数の実部と虚部は、それぞれ波形の横方向の運動と縦方向の運動に対応します。

複素数波​

オイラーの公式を使って、波形の式を改訂することができます。例えば、次の複素数波形を考えてみましょう：

z(t)=Aei(2πft+ϕ)z(t) = A e^{i(2\pi f t + \phi)}z(t)=Aei(2πft+ϕ)

ei(2πft+ϕ)e^{i(2\pi f t + \phi)}ei(2πft+ϕ) を展開し、オイラーの公式を使うと次のようになります：

z(t)=A[cos⁡(2πft+ϕ)+isin⁡(2πft+ϕ)]z(t) = A \left[\cos(2\pi f t + \phi) + i\sin(2\pi f t + \phi)\right]z(t)=A[cos(2πft+ϕ)+isin(2πft+ϕ)]

これにより、同一の波形が次のように表されることがわかります：

  • 実部 ℜ(z)=Acos⁡(2πft+ϕ)\Re(z) = A \cos(2\pi f t + \phi)ℜ(z)=Acos(2πft+ϕ)
  • 虚部 ℑ(z)=Asin⁡(2πft+ϕ)\Im(z) = A \sin(2\pi f t + \phi)ℑ(z)=Asin(2πft+ϕ)

したがって、正弦波と余弦波を複素数形式で表現する場合、複素数の実部と虚部を使って表すことができます：

  1. 正弦波 y(t)=Asin⁡(2πft+ϕ)y(t) = A \sin(2\pi f t + \phi)y(t)=Asin(2πft+ϕ)：

    • 複素数の虚部を使って表すと： y(t)=ℑ(Aei(2πft+ϕ))y(t) = \Im \left(A e^{i(2\pi f t + \phi)}\right)y(t)=ℑ(Aei(2πft+ϕ))

  2. 余弦波 y(t)=Acos⁡(2πft+ϕ)y(t) = A \cos(2\pi f t + \phi)y(t)=Acos(2πft+ϕ)：

    • 複素数の実部を使って表すと： y(t)=ℜ(Aei(2πft+ϕ))y(t) = \Re \left(A e^{i(2\pi f t + \phi)}\right)y(t)=ℜ(Aei(2πft+ϕ))

ここで、Aei(2πft+ϕ)A e^{i(2\pi f t + \phi)}Aei(2πft+ϕ) は統一された複素数表示であり、実部と虚部がそれぞれ余弦波と正弦波に対応しています。

波の重ね合わせ​

三角関数を用いて二つの波形を直接重ね合わせる場合、三角関数の加法定理を使用する必要があります。例えば、次のような式です：

cos⁡(A+B)=cos⁡Acos⁡B−sin⁡Asin⁡B\cos(A + B) = \cos A \cos B - \sin A \sin Bcos(A+B)=cosAcosB−sinAsinB

および

sin⁡(A+B)=sin⁡Acos⁡B+cos⁡Asin⁡B\sin(A + B) = \sin A \cos B + \cos A \sin Bsin(A+B)=sinAcosB+cosAsinB

二つの波形が重なる場合：

y1(t)=A1cos⁡(2πf1t+ϕ1),y2(t)=A2sin⁡(2πf2t+ϕ2)y_1(t) = A_1 \cos(2\pi f_1 t + \phi_1), \quad y_2(t) = A_2 \sin(2\pi f_2 t + \phi_2)y1​(t)=A1​cos(2πf1​t+ϕ1​),y2​(t)=A2​sin(2πf2​t+ϕ2​)

それぞれの波形を展開し、cos⁡\coscos と sin⁡\sinsin の加法定理を処理して、実部と虚部を整理する必要があります。

この過程は非常に煩雑で、特に波形の数が増えると、式の複雑さは指数関数的に増加します。

z1=A1ei(2πf1t+ϕ1),z2=A2ei(2πf2t+ϕ2)z_1 = A_1 e^{i(2\pi f_1 t + \phi_1)}, \quad z_2 = A_2 e^{i(2\pi f_2 t + \phi_2)}z1​=A1​ei(2πf1​t+ϕ1​),z2​=A2​ei(2πf2​t+ϕ2​)

ここで重要なのは、複素数の加法が「線形」であるという点です。つまり、二つの波形の振幅と位相は直接加算または分離して保持でき、三角関数の展開を使って処理する必要はありません。

一方で、複素数には実部と虚部が組み込まれており、これ自体が cos⁡\coscos と sin⁡\sinsin の情報を含んでいます。複素数加法では、実部と虚部がそれぞれ波形の余弦成分と正弦成分に対応するため、三角関数を手動で分解する必要はありません。

複素数形式のもう一つの利点は、波形の「振幅」と「位相」を明確に分離できることです：

  • 振幅は複素数の絶対値 ∣z∣|z|∣z∣ で決まります。
  • 位相は複素数の偏角 arg⁡(z)\arg(z)arg(z) で決まります。

波形の重ね合わせでは、これらの情報を個別に処理することも、まとめて計算に使うこともできます。例えば、重ね合わせた波形の振幅を簡単に分析したい場合、∣z∣=ℜ(z)2+ℑ(z)2|z| = \sqrt{\Re(z)^2 + \Im(z)^2}∣z∣=ℜ(z)2+ℑ(z)2​ を直接計算すればよいのです。

最後に、波形の位相回転は複素数の相角偏移に対応します。重ね合わせた後の波形の位相を調整する必要がある場合、eiθe^{i\theta}eiθ に回転角度 Δθ\Delta\thetaΔθ を加えるだけでよく、cos⁡\coscos と sin⁡\sinsin の位相を個別に調整する必要はありません。

フーリエ変換​

上で説明した内容は、フーリエ変換の数学的公式を理解するための準備に過ぎません。

フーリエ変換の核心となる概念は次の通りです：

  • 任意の信号は、正弦波と余弦波の重ね合わせとして表現できる。

これはつまり、たとえ信号が非常に複雑に見えても、例えば音楽の一節や画像、あるいはインパルス信号であっても、それを簡単な基本的な波形の組み合わせとして分解できるということです。これらの基本的な波形が、私たちがよく知っている正弦波と余弦波であり、これらが異なる周波数、振幅、位相で共同して信号の全体像を形成します。

正弦波と余弦波は非常に強力な数学的性質を持っています。周期的な現象に対して、正弦波と余弦波は「基底」として機能します。空間で位置を記述する際の x,y,zx, y, zx,y,z の座標軸のようなものです。適切な線形結合を通じて、私たちは任意の複雑な形状や変化を表現できます。

信号 y(t)y(t)y(t) が時間変化する関数であると仮定したとき、フーリエ変換は次の二つの重要な質問に答える手助けをします：

  1. この信号にはどんな周波数成分が含まれているのか？
  2. 各周波数の振幅と位相はどのくらいか？

この周波数分解により、信号を全く異なる角度から理解できるようになります。時間的に変化する波形を直接観察するのではなく、信号の周波数特性をより明確に見ることができます。

フーリエ変換の数学的定義は、信号を「時間領域」から「周波数領域」へ変換する方法を提供します。その核心となる公式は次の通りです：

Y(f)=∫−∞∞y(t)e−i2πft dtY(f) = \int_{-\infty}^\infty y(t) e^{-i 2\pi f t} \, dtY(f)=∫−∞∞​y(t)e−i2πftdt

この公式の各部分の物理的意味は次の通りです：

  1. 元の信号 y(t)y(t)y(t)：

    y(t)y(t)y(t) は時間領域で定義された関数です。これは音楽の音波、電気信号の振幅変化、または特定の時間内で変化するインパルスのシーケンスなど、あらゆる種類の信号であり得ます。

    y(t)y(t)y(t) は、信号の各時刻 ttt での振幅値を含んでいます。

  2. 複素指数関数 e−i2πfte^{-i 2\pi f t}e−i2πft：

    これはフーリエ変換の鍵となる部分で、実際には正弦波と余弦波の組み合わせです：

    e−i2πft=cos⁡(2πft)−isin⁡(2πft)e^{-i 2\pi f t} = \cos(2\pi f t) - i \sin(2\pi f t)e−i2πft=cos(2πft)−isin(2πft)

    フーリエ変換において、e−i2πfte^{-i 2\pi f t}e−i2πft は周波数 fff を持つ複素波形を用いて y(t)y(t)y(t) と一致させ、信号のその周波数成分の強度を抽出します。

積分操作は本質的に内積の計算であり、y(t)y(t)y(t) と e−i2πfte^{-i 2\pi f t}e−i2πft の類似度を測定します。

この「類似度」が、信号の周波数 fff における貢献度を決定します。

フーリエ変換は、すべての可能な周波数 fff に対してスキャンを行います。

各 fff に対して、積分計算によって得られた値 Y(f)Y(f)Y(f) がその周波数の強度を表します。これが「スペクトル」と呼ばれる理由です。

ここまで説明したところで、実際に自分で計算してみましょう：

単一周波数の信号を考えます：

y(t)=Acos⁡(2πf0t+ϕ)y(t) = A \cos(2\pi f_0 t + \phi)y(t)=Acos(2πf0​t+ϕ)

オイラーの公式を使って、これを次のように表現します：

y(t)=ℜ{Aei(2πf0t+ϕ)}y(t) = \Re\left\{ A e^{i (2\pi f_0 t + \phi)} \right\}y(t)=ℜ{Aei(2πf0​t+ϕ)}

この信号に対してフーリエ変換を行うと：

Y(f)=∫−∞∞y(t)e−i2πft dtY(f) = \int_{-\infty}^\infty y(t) e^{-i 2\pi f t} \, dtY(f)=∫−∞∞​y(t)e−i2πftdt

y(t)y(t)y(t) の表現を代入すると次のようになります：

Y(f)=∫−∞∞ℜ{Aei(2πf0t+ϕ)}e−i2πft dtY(f) = \int_{-\infty}^\infty \Re\left\{ A e^{i (2\pi f_0 t + \phi)} \right\} e^{-i 2\pi f t} \, dtY(f)=∫−∞∞​ℜ{Aei(2πf0​t+ϕ)}e−i2πftdt

フーリエ変換の核心は積分操作であり、その効果は時間領域上のすべての点と周波数波形を一致させることです。もし f=f0f = f_0f=f0​ ならば、e−i2πfte^{-i 2\pi f t}e−i2πft と ei2πf0te^{i 2\pi f_0 t}ei2πf0​t は完全に一致し、積分の結果はゼロではなくなります。

もし f≠f0f \neq f_0f=f0​ ならば、これらの波形は位相が一致せず、積分の結果はほぼゼロになります。

周波数が一致したとき、得られる結果は次のようになります：

Y(f0)=AeiϕY(f_0) = A e^{i \phi}Y(f0​)=Aeiϕ

これは、スペクトル Y(f)Y(f)Y(f) がその周波数の強度（AAA によって与えられる）だけでなく、その周波数の位相（ϕ\phiϕ によって与えられる）も含んでいることを示しています。

これが、フーリエ変換が信号を完全に記述できる重要な理由です。

フーリエ級数​

フーリエ級数はフーリエ変換の特別な場合であり、主に周期的な信号を扱うために使用されます。

フーリエ変換が信号の周波数スペクトルを連続的な周波数範囲に拡張できるのに対して、フーリエ級数は周期的信号を「離散的な周波数」を用いて表現することに焦点を当てています。

フーリエ級数の核心となる概念は次の通りです：

  • 任意の周期的信号は、「離散周波数」の正弦波と余弦波の線形結合として表現できる。

これは、信号が周期的であれば、限られた数の正弦波と余弦波を使ってその信号を近似でき、周波数の組み合わせが増えるにつれて、近似精度が高くなることを意味します。

周期 TTT を持つ信号 x(t)x(t)x(t) は、フーリエ級数で次のように表すことができます：

x(t)=a0+∑n=1∞[ancos⁡(2πntT)+bnsin⁡(2πntT)]x(t) = a_0 + \sum_{n=1}^\infty \left[ a_n \cos\left(\frac{2\pi n t}{T}\right) + b_n \sin\left(\frac{2\pi n t}{T}\right) \right]x(t)=a0​+n=1∑∞​[an​cos(T2πnt​)+bn​sin(T2πnt​)]

ここで、a0,an,bna_0, a_n, b_na0​,an​,bn​ はフーリエ級数の係数で、それぞれ異なる周波数に対応する正弦波と余弦波の振幅を示します。これらの係数は次の式で決定されます：

  1. 直流成分（a0a_0a0​）：

    a0=1T∫0Tx(t) dta_0 = \frac{1}{T} \int_{0}^T x(t) \, dta0​=T1​∫0T​x(t)dt

    これは信号の平均値、すなわち信号の全周期にわたる直流成分を表します。

  2. 余弦波係数（ana_nan​）：

    an=2T∫0Tx(t)cos⁡(2πntT) dta_n = \frac{2}{T} \int_{0}^T x(t) \cos\left(\frac{2\pi n t}{T}\right) \, dtan​=T2​∫0T​x(t)cos(T2πnt​)dt

    これは信号中の周波数 nT\frac{n}{T}Tn​ に対応する余弦波の振幅を表します。

  3. 正弦波係数（bnb_nbn​）：

    bn=2T∫0Tx(t)sin⁡(2πntT) dtb_n = \frac{2}{T} \int_{0}^T x(t) \sin\left(\frac{2\pi n t}{T}\right) \, dtbn​=T2​∫0T​x(t)sin(T2πnt​)dt

    これは信号中の周波数 nT\frac{n}{T}Tn​ に対応する正弦波の振幅を表します。

同様に、計算と表記を簡素化するために、フーリエ級数は複素数形式でも表されます：

x(t)=∑n=−∞∞cnei2πntTx(t) = \sum_{n=-\infty}^\infty c_n e^{i \frac{2\pi n t}{T}}x(t)=n=−∞∑∞​cn​eiT2πnt​

ここで、cnc_ncn​ は複素数の係数で、次のように定義されます：

cn=1T∫0Tx(t)e−i2πntT dtc_n = \frac{1}{T} \int_{0}^T x(t) e^{-i \frac{2\pi n t}{T}} \, dtcn​=T1​∫0T​x(t)e−iT2πnt​dt

この形式では：

  • 実部 ℜ{cn}\Re\{c_n\}ℜ{cn​} は ana_nan​ に対応します。
  • 虚部 ℑ{cn}\Im\{c_n\}ℑ{cn​} は bnb_nbn​ に対応します。

フーリエ級数は工学や科学の分野で広く応用されており、例えば機械構造における周期的振動の研究や、方形波、三角波、鋸歯状波などの周期的電気信号の分析などに使用されます。

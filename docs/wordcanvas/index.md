# WordCanvas

本專案是一款基於 Pillow 的文字圖像渲染工具，專為隨機圖像生成設計。

通過新增多樣化的參數設定功能，使用者可靈活調整輸入文字、字型樣式與顏色等屬性，實現大規模文字圖像的隨機生成。不論是解決資料不足、類別不平衡問題，還是提升圖像多樣性，WordCanvas 都能提供簡單高效的解決方案，為深度學習訓練提供堅實的資料基礎。

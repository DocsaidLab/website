centercrop

centercrop(img: np.ndarray) -> np.ndarray

  • 說明：對輸入影像進行中心裁剪處理。

  • 參數

    • img (np.ndarray)：要進行中心裁剪處理的輸入影像。

  • 傳回值

    • np.ndarray：裁剪後的影像。

  • 範例

    import capybara as cb

    img = cb.imread('lena.png')
    img = cb.imresize(img, [128, 256])
    crop_img = cb.centercrop(img)

    綠色框表示中心裁剪的區域。

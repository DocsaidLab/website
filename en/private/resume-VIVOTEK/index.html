Master’s Thesis​

  • Title: "Real-Time Error Correction of Hourly Rainfall Ensemble Forecasts"
  • Abstract: The Weather Research and Forecasting (WRF) model is commonly used in meteorology for rainfall forecasting, but its forecast results vary with model parameters, often leading to inaccurate estimates. Effective error correction of WRF ensemble forecast data could reduce forecast errors and improve model accuracy. This study used radar-estimated rainfall as observation values to calculate the error between WRF ensemble forecast data and observations. Two machine learning methods, Support Vector Machine (SVM) and Random Forest, were used to learn the error structure of WRF ensemble grid forecast rainfall data, establishing a real-time error correction model for rainfall forecast correction from 1 to 6 hours lead time. This research used two modeling approaches with machine learning: (1) Combining all ensemble members into a single error correction model (combined modeling), directly correcting ensemble mean values; and (2) creating an independent error correction model for each ensemble member (independent modeling), correcting each WRF ensemble member before averaging the corrected forecasts. Thus, four real-time error correction models were established: SVM with combined modeling, SVM with independent modeling, Random Forest with combined modeling, and Random Forest with independent modeling. Analysis showed that combined and independent modeling performed similarly, and combined modeling was recommended for simplicity and reduced computation time. Both machine learning methods significantly improved WRF ensemble rainfall forecasts for most typhoon events, especially for the 1 to 3 hours lead time, while improvement at 4 to 6 hours lead time was less pronounced. The proposed real-time error correction model effectively improved the accuracy of WRF ensemble forecasts, mitigating rainfall forecast underestimation; further analysis and comparison between the machine learning methods are suggested to explore differences in performance.

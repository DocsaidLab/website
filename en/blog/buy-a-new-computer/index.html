RTX 4090​

It all started here.

We first checked out and found that several manufacturers offer the 4090: ASUS, GIGABYTE, and MSI.

Next, we found a test video:

  • MSI RTX 4090 Suprim X vs. RTX 4090 Gaming X Trio, Thermals, Power & Overclocking

It looks like MSI runs particularly hot, while the other two brands have similar temperatures.

We also found an article:

  • [Info] 6 RTX 4090 Cooling Comparison

Overall, GIGABYTE seems to have the best performance, so we chose the best-looking version from their product line: the white Windforce edition.

Detailed specs:

  • Product: GIGABYTE RTX 4090 AERO OC 24G White Windforce Cooling Metal Backplate

  • Price: $60,900 NTD

  • Length: 34.2 cm

Motherboard​

With the GPU decided, the next step is the motherboard.

Here, we encountered the issue of socket compatibility. Different motherboards correspond to different CPUs, so we did some more research:

  • Intel 12th Gen pairs with 600 series motherboards using LGA 1700 socket.
  • Intel 13th Gen pairs with 700 series motherboards using LGA 1700 socket.
  • Intel 14th Gen pairs with 700 series motherboards using LGA 1700 socket.

It looks like the LGA 1700 socket is a longer-term choice, so we decided to go with the 700 series motherboard.

LGA 1700 Socket​

According to Wikipedia on LGA 1700:

LGA 1700 is a microprocessor socket used by Intel for Alder Lake (12th Gen Intel Core) and Raptor Lake (13th Gen, 14th Gen Intel Core) desktop CPUs. It replaces LGA 1200 and supports DDR5 memory. Due to the significant increase in contact points, previous coolers compatible with LGA 1200 or LGA 1151 (package size 37.5 mm x 37.5 mm) are incompatible with LGA 1700 (package size 45.0 mm x 37.5 mm).

CPU​

We’ve heard that Intel 13th Gen and 14th Gen have some quirks.

But since we’re not overclocking, it should be fine, so we decided to go with Intel 13th Gen.

Detailed specs:

  • Product: Intel i9-13900K 【24 Cores/32 Threads】3.0GHz-5.8GHz 36MB LGA1700 125W
  • Price: $19,500 NTD
tip

We run Ubuntu, and indeed encountered some crashes.

However, after a certain system update, the issue disappeared, so we're not sure whether to blame the CPU for a defect.

Memory​

The three major memory brands we know are Micron, Kingston, and Adata. Since we’re not professional gamers, we just picked from these three.

The key is capacity—since we’re training models, filling up 128GB is a basic requirement, right?

Detailed specs:

  • Product: Kingston 64 GB (Dual Channel 32GBx2) DDR5 6000 CL36 FURY Beast White

  • Price: $6,129 NTD / 1 Set

  • Quantity: 2 Sets

Storage​

For this part, we found a guide from a storage expert on PTT:

  • SSD (Solid State Drive) Buying Guide

There are many additional readings and comparisons of various brands in that guide, but the information was overwhelming.

We glimpsed a conclusion that recommended Micron, so we went with that! (Maybe too casually?)

Since we’re training models, we went with 4TB of storage to fit the datasets.

Detailed specs:

  • Product: Micron Crucial P5 Plus 2TB PCIe 4.0 NVMe SSD (Read 6600M / Write 5500M) TLC NAND

  • Price: $3,900 NTD / 1 Unit

  • Quantity: 2 Units

Liquid Cooling​

Here, the brands included ASUS, MSI, Corsair, and NZXT. Although there seemed to be many choices, we couldn’t find any particularly reliable reviews online, so we focused on warranty duration.

Most brands offer a 3-year warranty, but NZXT offers 6+1 years, so we went with NZXT.

Detailed specs:

  • Product: NZXT Kraken 360 RGB White

  • Price: $7,990 NTD

  • Thickness: 5.6 cm

tip

If you’re a professional in this area, you might have more expert advice for us, and we’ll consider it when building our next machine.
